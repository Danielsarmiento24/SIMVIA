# docs — Documentación técnica y académica

**Propósito:** Reunir diagramas, descripciones metodológicas, normas aplicables y referencias bibliográficas usadas en la tesis.

**Estructura:**
- `diagrams/` — Diagramas editables (drawio, SVG, PPTX) y PNG exportados.
- `methodology/` — Descripción del método, pipeline de desarrollo y criterios de validación.
- `standards/` — Legislación y normas técnicas (Ley 1581, normas de tráfico, estándares IoT).
- `references/` — BibTeX / PDFs relevantes y guía de citación.
