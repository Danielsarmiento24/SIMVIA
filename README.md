# hardware — Diseño y especificaciones electrónicas

**Propósito:** Documentar selección, esquemáticos, cálculos eléctricos, BOM y diseño mecánico del prototipo.

**Estructura:**
- `schematics/` — Esquemáticos eléctricos (KiCad/Altium/Eagle) y PDFs.
- `power/` — Cálculos de consumo, baterías, gestión de energía.
- `enclosure/` — Diseños CAD, consideraciones IP, ventilación y montaje.
- `datasheets/` — Datasheets referenciadas.
- `bill_of_material a/` — Detalles del hardware utilizado y precios aproximados.
