# Software — Código fuente y módulos

**Propósito:** Código modular para captura, inferencia, comunicación y APIs.

**Estructura:**
- `edge/` — Scripts para ejecutar en RPi (inference loop, init).
- `vision/` — Model wrappers, postprocessing, tracking, speed estimation.
- `tests/` — Pruebas unitarias y scripts de validación.
